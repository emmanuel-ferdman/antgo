======================
ANTGO
======================

目标
----------------------
ANTGO 提供了一个一站式AI模型训练、分析、部署、优化管理平台。


安装
----------------------
1. (RECOMMENDED)使用Docker环境

    docker pull registry.cn-hangzhou.aliyuncs.com/vibstring/antgo-env-dev:latest

2. install

    pip3 install antgo@git+https://github.com/jianzfb/antgo.git
